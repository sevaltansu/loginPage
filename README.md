# Giriş Ekranı Uygulaması Öğrenim Notları

Bu README dosyası, Giriş Ekranı Uygulaması'nı oluştururken edinilen tecrübe ve öğrenim notlarını içerir.

## Öğrenim Hedefleri

Bu projeyi yaparak aşağıdaki konularda öğrenim hedeflerine ulaşıldı:

 **Form Doğrulama:** Kullanıcının giriş yaparken girdiği verilerin doğrulanması için Form widget'ını nasıl kullanacağımı öğrendim. Validator fonksiyonlarını ve onSaved callback'lerini kullanarak giriş verilerini doğrulama sürecini oluşturdum.

 **Firebase Kimlik Doğrulama:** Firebase Authentication hizmetini kullanarak kullanıcıların giriş yapma işlemlerini nasıl yöneteceğimi öğrendim. createUserWithEmailAndPassword ve signInWithEmailAndPassword gibi işlevleri kullanarak kimlik doğrulama süreçlerini oluşturdum.

 **Hata Yakalama:** Firebase tarafından dönen hataları yakalayarak kullanıcı dostu hata mesajları oluşturma yeteneğini geliştirdim. Try-catch yapısını kullanarak hata durumlarıyla nasıl başa çıkabileceğimi öğrendim.

## Öğrenim Deneyimi

Bu projeyi yaparken, Flutter ve Firebase ile ilgili temel konuları daha iyi anlama fırsatım oldu. Özellikle kullanıcı girişi ve kimlik doğrulama gibi temel işlemleri nasıl gerçekleştireceğimi öğrenmek benim için önemliydi. Ayrıca form doğrulama, ekran yönlendirmeleri ve hata yakalama gibi konulara daha fazla hakim oldum.
